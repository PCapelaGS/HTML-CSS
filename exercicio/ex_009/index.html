Listas Ordenadas:

Para ordrnarmos as listas de forma mais fácil, e de forma que consigamos editalas sem problemas usamos a tag OL:

  1. Acordar
  2. Arrumar a cama
  3. Tomar café
  4. Escovar os dentes
  5. Se trocar

cada intem da lista é caracterizado pela tag LI, sendo o \LI opcinional nas linhas, tendo que ser usao apenas no final

Também mudar o tipo de lista, se queremos ela com números, letras, algarismo romano, etc. Para isso basta usar OL TYPE=""

Listas Não Ordenadas:

São aquelas na qaul a sequência não precisa ter sentido, normalmente usando símbolos para representação.

Para criarmos uma lista desse tipo usamos a tag UL

  • Carro
  • Tênis
  • Tattoo
  • Viajem

Este tipo de listas também possuem diversos tipos de estilo, sendo usados da mesma forma da anterioro UL TYPE="", vejamos a seguir:

Listas de Definição:

Funciona como um dicionário, dando a explicação do que é algo. Para ele usamos a tag DL com os itens sendo caracterizado poe DT e seu significado DD. Vejamos o exemplo:

Haule:
É aquele que não é do local, um estrangeiro, alguém de fora.
Prego:
Pessoa que não sabe fazer algo, se acha mas faz tudo errado.
Vala/Pico:
Pode ser definido como local do surf.
Nativo:
Pessoa na qual sempre surfa naquele local.
Quebrar:
Quando alguém esta surfando muito bem
