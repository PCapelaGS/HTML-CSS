Usando Links externos

Links externos são aqueles que ligam nosso site a outros, nos encaminhando para ele.

Para usarmos links externos basta adicionarmos a tag A, sempre é recomenrdado usar o TARGET="_BLANK" REL="EXTERNAL", vejamos no exemplo:

Você pode acessar os jogos de hoje no site do Globo Esporte

Você também pode acessar outros sites, como o Youtube, para ver os melhores momentos da partida

Usando Links internos:

Este tipo de link serve para fazer a relação a uma página que está dentro de nosso próprio servidor.

A tag para sua utilização é a mesma de Links Externos, apenas muda que usaremos a URL de uma página que já possuimos. Vejemos o exemplo:

Esta é a primeira página do meu site, caso seja de seu interesse você tambem pode acessar minha segunda página

Você também pode acessar a nossa páginas de notícias.
