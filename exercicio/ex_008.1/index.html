Outras formatações:

Código-fonte:

São partes de comandos, por exemplo, o comando document.getElementyById('teste') é escrito em linguagem JavaScript. Para isso envelopamos o código com tag CODE

Agora um exemplo na Linguagem Python: 

            
                num = int(input('Digite um número:'))
                if num % 2 == 0:
                    print(f'O número {num} é PAR)
                else:
                    print(f'O número {num} é IMPAR)
                print('Fim do programa')

O CODE não salva seu código com formatação. Para formatarmos precisamos usar a tag PRE antes da tag CODE, como podemos ver a cima.

Citações Simples:

Para fazermos a citação de alguma frase basta usarmos a tag Q, vejamos a seguir:

Como já diria seu: A vingança nunca é plena, mata a alma e a invenena

Citações compostas:

Para usarmoscitações em bloco usamos a tag BLOCKQUOTE, vamos ver em prática:

Segundo Charles Darwin, no seu livro A Origem das Espécies:

De um começo tão singelo evoluíram as formas mais belas e maravilhosas

Alem disso conseguimos referênciar o livro do qual foi tirado o texto através da tag BLOCKQUOTE CITE=""

Abreviações (abbr):

Para fazermos abreviações usamos a tag ABBR, indicando que é uma abreviação. Por exemplo:

Estou estudando HTML e CSS, e estou gostando muito.

Texo Invertido (Texto Invertido:):

Também conseguimos criar TEXTOS INVERTIDOS usando a tag BDO + o sentido de escrita (rtl ou ltr), assim a tag fica BDO DIR="rtl ou ltr"; vejamos um exemplo:

Meu nome é Pedro, se escrevermos ele invertido fica Pedro
